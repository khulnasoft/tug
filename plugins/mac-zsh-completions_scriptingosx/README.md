> macOS zsh Completions

Some zsh completion files for macOS specific commands and third party tools including

* PlistBuddy
* bbedit
* desktoppr
* gh
* installer
* jamf
* plutil
* project
* see
* softwareupdate
* swift
* xcrun
* xed
